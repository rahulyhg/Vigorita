Create interactive 'shoppable' images with click-to-buy tags. Ideal to showcase your products in a different way and drive more traffic.

== Description ==

Increase traffic through your website by creating shoppable images: images containing different hotspots which link to various products or pages.

= Features =

* Woocommerce 3.0+ compatible
* Create shoppable images in 3 easy steps, straight from the WordPress admin
* Change tag colors
* Easily link hotspots to your productsÂ or any page
* Add shoppable images on posts or pages by using shortcodes
* Or add it anywhere in your theme through PHP.
